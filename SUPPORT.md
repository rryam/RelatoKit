# Support

Use GitHub Issues for reproducible bugs and concrete feature requests.

Please include:

- macOS version
- Xcode command line tools version
- `relato version`
- the command you ran
- whether Feedback Assistant is signed in
- whether Terminal has Accessibility permission, if using `fill` or `submit`

Do not include private Apple account identifiers, team IDs, unpublished feedback contents, or attachments that contain confidential project data.
